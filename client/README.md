# WanderWise — Frontend

React + Vite client for **WanderWise**, a smart travel planning platform.

📖 Full documentation, features and setup: [main README](../README.md)

## Quick Start

    npm install
    npm run dev

Runs at **http://localhost:5173** — requires the [backend](../server) to be running.

## Stack
React 18 · Vite · React Router · Axios · modular plain CSS
